A fun side-scrolling flying game featuring a glowing ghost character navigating through obstacles across 5 challenging levels.

## Features

- **Beautiful Visuals**: Animated ghost with glowing effects, particle trails, and dynamic backgrounds
- **5 Progressive Levels**: From Peaceful Clouds to Nightmare Realm
- **Dynamic Difficulty**: Each level increases speed, gravity, and reduces obstacle gaps
- **Particle Effects**: Trail particles and jump burst effects
- **Level Transitions**: Smooth "LEVEL UP!" animations between levels

## Levels

1. **Peaceful Clouds** (0-9 points) - Light blue sky, easy start
2. **Windy Skies** (10-19 points) - Deeper blue, slightly faster
3. **Storm Clouds** (20-29 points) - Lightning effects, darker atmosphere
4. **Lightning Zone** (30-39 points) - Pulsing obstacles, challenging gaps
5. **Nightmare Realm** (40+ points) - Dark purple/red theme, maximum difficulty

## Controls

- **SPACE** - Make the ghost fly up
- **R** - Restart after game over

## Installation

1. Make sure you have Python installed
2. Install pygame:
   ```bash
   pip install pygame
   ```
3. Run the game:
   ```bash
   python ghost_flyer.py
   ```

## How to Play

- Press SPACE to make the ghost fly upward
- Navigate through gaps in the obstacles
- Each obstacle you pass gives you 1 point
- Every 10 points advances you to the next level
- Try to reach the Nightmare Realm!
